# springboot-docker-sample

## Things to do:
* Clone Repository: 
    ```
    git clone https://github.com/hendisantika/springboot-docker-sample.git
    ```
* Go to the folder:
    ```
    springboot-docker-sample
    ```
* Package the app:
    ```
    mvn clean package
    ```
* Run docker-compose:
    ```
    docker-compose --compatibility up -d
    ```
* Check the docker images
    ```
    docker ps
    ```
